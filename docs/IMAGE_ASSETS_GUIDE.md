## Image Optimization Tips

### For Mobile App
- Resize to max 1080px width
- Compress JPEGs to 60-80% quality
- Use WebP format if possible
- Keep file size under 200KB per image

### For Faded Backgrounds
- Apply 10-20% opacity
- Use blur effect (5-10px radius)
- Desaturate colors slightly
- Test readability of text overlay

### For Category Cards
- Apply color overlay matching category
- Use 50-70% overlay opacity
- Ensure contrast with text
