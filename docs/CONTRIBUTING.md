# Contributing to VAZHI

நன்றி! Thank you for your interest in contributing to VAZHI.

## Ways to Contribute

### 1. Data Contributions

We need high-quality Tamil text data:

- **Conversational pairs**: Natural Tamil dialogues
- **Q&A pairs**: Questions and answers in Tamil
- **Domain-specific**: Legal, government, education content
- **Scam examples**: Tamil fraud messages for security pack

**Format:**
```json
{
  "instruction": "Your question in Tamil",
  "output": "Expected answer in Tamil"
}
```

### 2. Pack Development

Create specialized LoRA packs:

1. Fork the repository
2. Create your pack in `vazhi-packs/your-pack-name/`
3. Include training data, scripts, and documentation
4. Submit a PR for review

### 3. Testing

- Test the model with diverse Tamil inputs
- Report issues with specific examples
- Test on different mobile devices

### 4. Documentation

- Translate docs to Tamil
- Write tutorials
- Improve code comments

## Code of Conduct

- Be respectful and inclusive
- Focus on helping Tamil speakers
- No proprietary data without permission
- Credit original sources

## Getting Started
