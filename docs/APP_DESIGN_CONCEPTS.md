# VAZHI App - Visual Design Concepts

## Current State Analysis

The current VAZHI app uses:
- **Primary**: `#FF6B35` (Saffron/Orange)
- **Secondary**: `#1E3A5F` (Deep Blue)
- **Background**: `#F5F5F5` (Light Gray)
- **Clean Material 3 design** but feels plain and generic

**Goal**: Add visual richness through category-specific imagery while maintaining accessibility and Tamil cultural identity.
